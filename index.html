What's Border ?

Border is an open source iframe Web browser developped by Onofficiel accessible on all platforms. Because of the iframe system some website won't work in this browser (like youtube.com). You can always use a x-frame killer like HiFrame.

Why use Border ?

Here are some features that Border has :
  • Multi Window System
  • User friendly UI
  • Ultimate customisation
  • Integrated Google support
  • A great community
  • Modern and minimalist style
  • Theme support
  • Extention support support
  • An API for devs
